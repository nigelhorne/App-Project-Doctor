  230: =head1 NAME
  231: 
  232: App::Project::Doctor::Check::Dependencies - Check that used modules are declared
  233: 
  234: =head1 DESCRIPTION
  235: 
  236: Scans all C<.pm>, C<.pl>, and script files for C<use>/C<require> statements
  237: and compares against C<cpanfile> or C<Makefile.PL> prerequisites
  238: (via L<App::makefilepl2cpanfile>).  Core modules are excluded.
  239: 
  240: =head3 MESSAGES
  241: 
  242:   Code | Trigger                      | Resolution
  243:   -----|------------------------------|-------------------------------------------
  244:   D001 | No builder or cpanfile found | Add a Makefile.PL or cpanfile
  245:   D002 | Module used but not declared | Fix appends a 'requires' line to cpanfile
  246: 
  247: =head3 FORMAL SPECIFICATION
  248: 
  249:   used     = { mod | (use|require mod) in source_files ctx }
  250:   declared = parse_prereqs (builder_file ctx)
  251:   missing  = used \\ (declared union CORE)
  252: 
  253:   check ctx ==
  254:     if declared = undef then [warning]
  255:     else [error+fix per m in missing] ++ (if missing = {} then [pass] else [])
